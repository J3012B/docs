---
title: 'Create Event'
openapi: 'POST /api/scheduled_events'
---

<Warning>
	A status code of 200 does not imply sucessful scheduling, but rather that the request was received successfully.
	For reliable scheduling completion info, please use the `webhook_url` field in the request to receive a webhook on completion.
</Warning>

<Note>
  Note: Currently the API only supports Calendly event types set to English, German, French and Spanish. Please [reach out](mailto:josef@trysetter.com), if you'd like us to support more languages. We can add more languages if there is a demand.
</Note>
